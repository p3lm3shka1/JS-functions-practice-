## [Live website](https://p3lm3shka1.github.io/JS-functions-practice-/)

### This project is a small JavaScript practice playground where multiple core JS concepts are implemented using simple UI elements.
Each block demonstrates a different feature: DOM manipulation, events, conditions, counters, text styling, calculations, and theme switching.
The goal of this project is learning and practicing JavaScript fundamentals, not visual perfection.

## 1️) Number Increment Counter
### A button that increases a number displayed on the screen.
- ```addEventListener```
- Variables
- DOM text update ```textContent```
- Basic state handling

## 2️) Even / Odd Number Checker
### User enters a number and clicks a button to check whether the number is even or odd.
- User input handling
- Type conversion
- Conditional logic ```if``` / ```else```
- Module operator (%)

## 3️) Forward / Backward Counter
### Two buttons allow increasing or decreasing a number within a fixed range (1–5).
- Multiple event listeners
- Boundary checks
- Preventing invalid state
- Separating logic from DOM

## 4️) Simple BMI Calculator
### User enters height (cm) and weight (kg), then calculates Body Mass Index (BMI).
- Mathematical calculations
- Input validation
- Working with numbers from text inputs
- Displaying dynamic results

## 5️) Text Zoom (Font Size Toggle)
### A button increases and decreases text size for multiple elements at once.
- ```querySelectorAll```
- Working with CSS classes
- ```classList.toggle```
- Styling multiple elements via JavaScript

## 6️) Font Style Controller
### Change font size using a number input
### Switch between normal, bold, and italic text styles
- input events
- Inline style manipulation
- Active button states
- Reusable functions

## 7️) Theme Switcher (Light / Dark)
### A toggle switch and button to change page themes.
- ```classList.add``` / remove / toggle
- Thme state handling
- CSS transitions
- Checkbox-based UI switch


## What Was Used
- HTML5
- CSS3
- JavaScript
- No external libraries or frameworks were used.
